# Internet-Independent Smart Wheelchair

This project implements a smart wheelchair system that works without internet
connectivity using ESP32 microcontrollers.

## Features
- Eye blink control
- Joystick control
- Voice command control
- Touchpad control
- Obstacle detection for safety
- ESP-NOW based offline communication

## Technologies Used
- ESP32
- Arduino IDE
- Embedded C
- Sensors and motor drivers

## Project Motivation
This project was built to help people with physical disabilities control a
wheelchair independently using multiple input methods.
